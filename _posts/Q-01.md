Given a non-empty array of integers, every element appears twice except for one. Find that single one.

Ex 1: 
```javascript
Input: [2,2,1]
Output: 1
```

Ex 2: 
```javascript
Input: [4,1,2,1,2]
Output: 4
```


​```javascript
var singleNumber = function(nums) {
  nums.sort()

  for (let i=0; i<nums.length; i += 2) {
    if (nums[i] !== nums[i + 1]) return nums[i]
  }
};
​```
